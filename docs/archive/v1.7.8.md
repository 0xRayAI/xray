## 📝 Migration Guide

### For Developers

**If upgrading from v1.7.2**:

1. **Update dependencies**:
   ```bash
   npm update
   ```

2. **Review routing config**:
   - Check `.opencode/strray/routing-mappings.json`
   - Ensure custom mappings are compatible

3. **Update analytics usage**:
   ```typescript
   // Old way (deprecated)
   const patterns = router.getEmergentPatterns();

   // New way (recommended)
   const summary = router.getDailyAnalyticsSummary();
   const analytics = router.getRoutingAnalytics();
   ```

4. **Update type imports**:
   ```typescript
   // Import new interfaces
   import {
     EmergentPattern,
     PatternUpdate,
     AdaptiveThresholds,
   } from './core/kernel-patterns';
   ```

### For Users

**Upgrading from v1.7.2**:

1. **Backup your configuration**:
   ```bash
   cp -r .opencode ~/.stringray-backup-1.7.2
   ```

2. **Update 0xRay**:
   ```bash
   npm update strray-ai@latest
   ```

3. **Review analytics settings**:
   - Check consent preferences
   - Review anonymization settings
   - Adjust pattern detection thresholds

4. **Test routing**:
   - Verify `@agent` names work correctly
   - Test custom keyword mappings
   - Review confidence scores
