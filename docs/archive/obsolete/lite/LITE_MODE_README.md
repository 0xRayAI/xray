# 0xRay Lite Extension

0xRay Lite provides essential AI-assisted development capabilities within the OpenCode ecosystem, optimized for teams that need quality safeguards without full framework complexity.

## Relationship to Full Framework

- **Base**: Built on OpenCode v1.1.1 (same as full version)
- **Agents**: 4 core agents (vs 8 in full version)
- **Configuration**: Uses same OpenCode.json schema
- **Models**: Same "openrouter/xai-grok-2-1212-fast-1" integration
- **Upgrade Path**: Can migrate to full framework when needed
