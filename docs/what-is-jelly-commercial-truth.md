## What Jelly Actually Is

Jelly is a **Project Intelligence Dashboard**.

It takes the 15 MCP servers from StringRay and presents them as a unified project management interface.

**Jelly does:**
1. **Auto-discovers** your project structure using librarian
2. **Analyzes complexity** using the orchestrator  
3. **Suggests agents** based on task type (architect for design, enforcer for compliance)
4. **Tracks progress** across all StringRay operations
5. **Generates reports** from the autonomous reporting system

**Jelly is the GUI that makes StringRay accessible to non-technical users.**
