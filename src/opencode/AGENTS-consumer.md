# xray 2.0 Consumer Note

This file previously contained extensive documentation for an earlier version of the framework (published under different names).

For current usage with **xray 2.0**:

- Agents are defined exclusively in `.opencode/agents/*.yml`
- The YML files are the single source of truth for agent capabilities
- See the main [AGENTS.md](../AGENTS.md) and [README.md](../README.md) in the package root for quick start

Legacy configuration paths (`.strray/`, `.opencode/strray/`) are no longer used. All new configuration lives under `.opencode/`.

This document is retained only as a minimal redirect for anyone who previously relied on the old consumer guide. No new content will be added here.
