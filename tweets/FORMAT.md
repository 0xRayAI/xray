# Tweet Format (0xRay releases)

## Template

```
🎉 0xRay v{VERSION} is LIVE — {THEME}!

✨ {feature 1}
🔌 {feature 2}
✅ {test count} tests passing

```
npm install 0xray@latest
```

{one-line pitch}

#0xRay #AIOps #DevTools #MCP #NPM
```

## Rules

1. **Header**: `🎉 0xRay vX.Y.Z is LIVE — {Theme}!`
2. **Bullets**: 3–6 lines with emoji prefixes (✨ 🔌 🧠 📚 ✅)
3. **Code block**: always `npm install 0xray@latest`
4. **Package name**: `0xray` (not `strray-ai` or `StringRay`)
5. **Counts**: use current verified numbers (agents, MCP servers, tests)
6. Save to `tweets/v{VERSION}.md`
