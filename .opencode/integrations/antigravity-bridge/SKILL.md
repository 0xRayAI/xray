# @antigravity-bridge Skill

## Purpose

Bridge skill that provides unified access to all Antigravity skills with improved UX and discovery.

## Skills Available via Bridge

| Skill | Purpose |
|-------|---------|
| `typescript-expert` | TypeScript patterns and best practices |
| `react-patterns` | React component patterns |
| `python-patterns` | Python coding patterns |
| `docker-expert` | Docker and containerization |
| `api-security-best-practices` | API security guidelines |
| `rag-engineer` | RAG architecture patterns |
| `seo-fundamentals` | SEO best practices |
| `prompt-engineering` | Prompt crafting techniques |
| `brainstorming` | Creative brainstorming |
| `copywriting` | Content writing guidelines |
| `pricing-strategy` | Pricing models and strategy |
| `vercel-deployment` | Vercel deployment guides |
| `aws-serverless` | AWS Lambda and serverless |

## Usage

```
@antigravity list          # List all available skills
@antigravity search <term> # Search skills by topic
@antigravity use <skill>  # Activate a specific skill
```

## Integration

This bridge is auto-installed by `npx strray-ai install` to provide unified access to all Antigravity skills.

## Update

```bash
npx strray-ai update-skills
```
