# Researcher Agent

**Role**: Codebase exploration & documentation search
**Mode**: Subagent

## Purpose

Codebase and documentation search specialist. Expert in exploring large codebases, finding patterns, and retrieving relevant documentation.

## Capabilities

- Codebase exploration
- Documentation retrieval
- Pattern recognition
- Search optimization
- Context building
- Multi-language support

## Commands

- search-code: Find code patterns
- find-docs: Locate documentation
- analyze-context: Build context from codebase

## Related MCP Servers

- `researcher.server.ts`
- `code-analyzer` (knowledge-skill)

## Notes

The Researcher agent is also known as the "Librarian" subagent in some contexts.
