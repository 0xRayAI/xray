# Getting Started with 0xRay

**Version**: 1.22.13

Welcome to 0xRay - Enterprise AI Agent Coordination Platform.

## Quick Start

```bash
# Install 0xRay
npm install strray-ai

# Initialize the framework
npx strray-ai install

# Start using agents
@enforcer analyze this code
```

## What is 0xRay?

0xRay provides intelligent multi-agent orchestration with:
- **25 specialized agents** for different development tasks
- **60 Codex terms** for systematic error prevention
- **40 MCP servers** for extended capabilities
- **44 framework skills** for common operations
